# Bordered Teacher-Mothers

Static ACWS 2026 presentation deck for Zhuo Chang's panel paper.

- GitHub Pages: https://changzhuo0200.github.io/Bordered-Teacher-Mothers/
- Netlify mirror: https://bordered-teacher-mothers.netlify.app/

The GitHub Pages version uses a lightweight `index.html` loader plus text chunks under `chunks/` so the deck can be maintained through the GitHub App text-file API.
